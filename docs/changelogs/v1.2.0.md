# v1.2.0 - Skill Renames, Library-Skill Expansion & Cross-Link Skill

**Date:** 2026-04-14

This release renames `agent-architect` → `agent-builder`, expands `library-skill` to maintain `docs/prd/PRD.md` and `docs/architecture.md`, adds the new `cross-link-skills` skill, and propagates rename references across the repo.

---

## Added

- **`cross-link-skills`** — new meta skill that repairs and verifies cross-references between SKILL.md files after structural changes (creation, rename, removal).

## Changed

- **`agent-architect` renamed to `agent-builder`** — directory and all internal/external references updated.
- **`library-skill`** — major expansion:
  - Now maintains `docs/architecture.md` (step 6) and `docs/prd/PRD.md` (step 7) in addition to SKILL-INDEX, AGENTS.md, README, and skill graph.
  - If `docs/architecture.md` does not exist, invokes `codebase-understanding` to create it.
  - If `docs/prd/PRD.md` does not exist, invokes `prd-writing` to create it — fed with architecture.md, existing point-in-time PRDs (treated as directional input, not ground truth), and the skill registry. Allows user clarifying questions.
  - Architecture step runs before PRD step so PRD creation can use the architecture output.
  - Step count increased from 8 to 10.
  - Living PRD maintained at `docs/prd/PRD.md`.
- **`ROUTING.md`** — all `agent-architect` references updated to `agent-builder`.
- **`create-agent-prompt`** — updated caller reference from `agent-architect` to `agent-builder`.
- **`improve-skills`** — minor structural edits.
- **`setup-evaluation`** — updated references from `agent-architect` to `agent-builder`.
- **`skill-finder`** — updated reference from `agent-architect` to `agent-builder`.
- **`tool-finder`** — minor reference update.
- **`universal-skill-creator`** — minor structural edits.
- **`docs/architecture.md`** — updated to reflect `agent-builder` rename.
- **`docs/changelogs/v1.1.0.md`** — minor correction.
