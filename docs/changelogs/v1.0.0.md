# v1.0.0 — First Release

**Date:** 2026-04-07

Write once, run everywhere. 37 agent skills that work across Codex CLI, Ampcode, Claude Code, Warp, Gemini CLI, GitHub Copilot, Cursor, VS Code, Factory.ai, Replit, and Bolt.new — following the [agentskills.io](https://agentskills.io/specification) open standard.

## Installation

# Windows (PowerShell)
git clone https://github.com/dvy1987/cross-plat-skills.git $HOME\.cross-plat-skills

## Key Design Decisions

- **200-line limit** on all skills — backed by SkillReducer research (arXiv:2603.29919) showing diminishing returns past 200 lines
- **Symlinks over copies** — `git pull` instantly updates all tools, no reinstall needed
- **Link-before-split** — when a skill grows too large, reuse an existing skill before creating a new one
- **Prune before research** — remove wrong content before adding new content

## Known Limitations

- Replit and Bolt.new require per-project copies — no global install path
