# v1.0.0 — First Release

**Date:** 2026-04-07

Write once, run everywhere. 37 agent skills that work across Codex CLI, Ampcode, Claude Code, Warp, Gemini CLI, GitHub Copilot, Cursor, VS Code, Factory.ai, Replit, and Bolt.new — following the [agentskills.io](https://agentskills.io/specification) open standard.

### Installation

One command, works globally across every tool:

```bash
# macOS / Linux
git clone https://github.com/dvy1987/cross-plat-skills.git ~/.cross-plat-skills
bash ~/.cross-plat-skills/install.sh

# Windows (PowerShell)
git clone https://github.com/dvy1987/cross-plat-skills.git $HOME\.cross-plat-skills
& $HOME\.cross-plat-skills\install.ps1
```

### Uninstallation

Clean removal that only removes repo-managed symlinks — user-owned skills are never touched:

```bash
bash ~/.cross-plat-skills/uninstall.sh            # macOS / Linux
.\uninstall.ps1                                    # Windows
```

Both support `--dry-run` / `-DryRun` to preview changes.

## Key Design Decisions

- **200-line limit** on all skills — backed by SkillReducer research (arXiv:2603.29919) showing diminishing returns past 200 lines
- **Symlinks over copies** — `git pull` instantly updates all tools, no reinstall
- **Link-before-split** — when a skill grows too large, check if an existing skill can absorb the sub-capability before creating a new one
- **Prune before research** — remove wrong content before adding new content
- **Security skills are immutable** — only human-authored commits, never compressed
