# v1.2.0 - Skill Renames, Library-Skill Expansion & Cross-Link Skill

**Date:** 2026-04-14

This release renames `agent-architect` ‚Üí `agent-builder`, expands `library-skill` to maintain `docs/PRD.md` and `docs/architecture.md`, adds the new `cross-link-skills` skill, and propagates rename references across the repo.

---

## Added

- **`cross-link-skills`** ‚Äî new meta skill that repairs and verifies cross-references between SKILL.md files after structural changes (creation, rename, removal).

## Changed

- **`agent-architect` renamed to `agent-builder`** ‚Äî directory and all internal/external references updated.
- **`library-skill`** ‚Äî major expansion:
  - Now maintains `docs/architecture.md` (step 6) and `docs/PRD.md` (step 7) in addition to SKILL-INDEX, AGENTS.md, README, and skill graph.
  - If `docs/architecture.md` does not exist, invokes `codebase-understanding` to create it.
  - If `docs/PRD.md` does not exist, invokes `prd-writing` to create it ‚Äî fed with architecture.md, existing point-in-time PRDs (treated as directional input, not ground truth), and the skill registry. Allows user clarifying questions.
  - Architecture step runs before PRD step so PRD creation can use the architecture output.
  - Step count increased from 8 to 10.
  - PRD and architecture paths now point to `docs/` directly (no sub-folders).
